# Estima Crédit - Simulateur de Prêt Intelligent

## Description

**Estima Crédit** est un simulateur de prêt en ligne intelligent conçu pour aider les utilisateurs à calculer et à planifier leurs emprunts financiers. L'application permet de simuler différents types de prêts (personnel, immobilier, auto, etc.), en tenant compte des facteurs comme le montant, la durée, le salaire mensuel et les taux d'intérêt. Elle offre une visualisation des résultats via des graphiques et des conseils éthiques inspirés des principes islamiques pour éviter le riba (intérêt) avec des consseil avant le validation des consseil d'or d'aprés[ايات من القران واحاديث من الرسول].

Le projet est développé en HTML, CSS (avec Tailwind CSS), et JavaScript, avec des bibliothèques externes comme Chart.js pour les graphiques. Il est entièrement responsive et adapté aux appareils mobiles et desktop.

## Fonctionnalités

- **Simulation de prêt** : Calcul automatique des mensualités, intérêts totaux, et montant total à rembourser.
- **Évaluation de la capacité financière** : Indicateur visuel (barre de progression) pour vérifier si l'emprunt est viable en fonction du salaire.
- **Visualisation des données** : Graphique circulaire (pie chart) pour représenter la répartition entre capital et intérêts.
- **Conseils éthiques** : Page dédiée avec des versets coraniques et hadiths sur le riba, pour sensibiliser aux risques spirituels.
- **Navigation fluide** : Menu responsive avec liens vers les pages Accueil, Simulation, Résultats, À propos, et Conseils d'Or.
- **Stockage local** : Utilisation de `localStorage` pour transférer les données de simulation entre les pages sans base de données.

## Technologies Utilisées

- **HTML5** : Structure des pages.
- **CSS3** : Styles avec Tailwind CSS pour la responsivité et les animations.
- **JavaScript (ES6)** : Logique de calcul, interactions, et manipulation du DOM.
- **Bibliothèques externes** :
  - [Tailwind CSS](https://tailwindcss.com/) : Framework CSS utilitaire.
  - [Chart.js](https://www.chartjs.org/) : Pour les graphiques.
  - [Font Awesome](https://fontawesome.com/) : Icônes.
  - [Google Fonts](https://fonts.google.com/) : Polices (Poppins, Amiri).
- **Outils** : Aucun serveur requis ; fonctionne en local ou sur un hébergement statique.

## Prérequis

- Un navigateur web moderne (Chrome, Firefox, Safari, Edge) supportant HTML5, CSS3, et JavaScript.
- Connexion Internet  .
- Aucun logiciel supplémentaire n'est nécessaire ; le projet est statique.
